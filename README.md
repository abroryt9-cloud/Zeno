# Zeno

Zeno is a lightweight messenger UI (Telegram-like) with:

- SMS login via Firebase Auth (Phone Number)
- Real-time chat via Firestore

## Policies

- [Privacy Policy](./privacy.html)
- [Terms of Use](./terms.html)

## Notes

To enable Phone Auth on the web (especially on GitHub Pages), configure your Firebase project:

- Enable Phone Sign-In in Firebase Authentication
- Add your GitHub Pages domain to **Authorized domains**
